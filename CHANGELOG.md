# Trulicare Project Changelog

## [1.0.0] - 2026-07-21
### Added
- Multi-timezone automated pre-alarm cron service.
- AES-256 PII field encryption for phone numbers and caregiver emails.
- Role-based access control (`restrictTo`) for pharmacist refill queues.
- Vite Rollup `manualChunks` code-splitting configuration.
- PDF clinical report generation via in-memory buffer streaming.
