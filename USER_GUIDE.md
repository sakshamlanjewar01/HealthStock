# Trulicare End-User & Role Guide

## 👤 Patient Guide
1. **Medication Management**: Navigate to Inventory to add prescriptions, set daily reminder schedules, and configure stock refill alerts.
2. **Dose Logging**: Record taken, skipped, or missed doses directly from the Dashboard or Calendar view.
3. **Clinical PDF Export**: Download adherence summaries for prescribing physicians.

## ⚕️ Pharmacist Guide
1. **Refill Queue**: View pending refill orders submitted by patients.
2. **Order Processing**: Update status to `Fulfilled` or `Rejected`.
