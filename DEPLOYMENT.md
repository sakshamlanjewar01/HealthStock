# Trulicare Deployment Guide

## 🚀 Production Deployment Stack

- **Frontend**: Vercel Edge CDN (Vite 5 Single Page Application)
- **Backend**: Render Container Instances (Node.js Express 4 REST API)
- **Database**: MongoDB Atlas Enterprise Cluster

---

## 🛠️ Build Commands

### Frontend Build
```bash
cd frontend
npm run build
```

### Backend Execution
```bash
cd backend
npm start
```
